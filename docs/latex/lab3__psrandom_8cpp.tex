\doxysubsection{Подробное описание}
Лабораторная работа №3. Исследование генераторов псевдослучайных чисел. 

Программа реализует три пользовательских генератора псевдослучайных чисел, сравнивает их статистические характеристики, выполняет упрощённые NIST-\/подобные тесты и измеряет скорость генерации. Результаты сохраняются в CSV-\/файлы для последующего построения графиков и включения в отчёт Doxygen.
