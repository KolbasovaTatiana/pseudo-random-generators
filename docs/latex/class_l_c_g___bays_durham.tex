\doxysubsection{Подробное описание}
Генератор на основе линейного конгруэнтного метода с перемешиванием Bays-\/\+Durham. 

Базовая последовательность получается линейным конгруэнтным генератором. Далее используется таблица перемешивания из 32 элементов для уменьшения корреляций.
